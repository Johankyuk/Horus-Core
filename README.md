# Horus

> **Nota:** Horus dejó de ser un proyecto pensado para todo el mundo.

Parte de lo que fue este proyecto —el asistente de instalación con múltiples modos, el selector de temas de color, el recoloreo de carpetas de Thunar y el soporte bilingüe— se archivó por ser demasiado demandante de mantener. Era trabajo que crecía más rápido de lo que podía sostenerlo en solitario.

A partir de ahora **solo doy soporte a la versión que uso día a día**: instalación simple, sin preguntas, en español y con tema morado fijo. Es mi setup personal, no un producto para distribuir.

El estado anterior (wizard, multi-tema, folders) queda congelado en la rama `archive/wizard-temas-folders` por si algún día lo retomo.

**La documentación real está en español:** [README.es.md](README.es.md)
